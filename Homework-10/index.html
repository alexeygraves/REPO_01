Кнопки — Button БЭМ: .btn

Три варианта: Primary, Secondary, Disabled. Три размера: sm, md, lg. Поддержка состояний hover, active, focus-visible.

Варианты

Размеры — Primary

Размеры — Secondary

<!-- Primary -->
<button class="btn btn--primary btn--md">Primary</button>

<!-- Secondary -->
<button class="btn btn--secondary btn--sm">Secondary</button>

<!-- Disabled -->
<button class="btn btn--primary btn--md" disabled>Disabled</button>

Поле ввода — Input БЭМ: .input-field

Состояния: normal, focus (кликните по полю), error. Поддержка иконки слева.

Введите корректный адрес электронной почты 
<!-- Normal -->
<div class="input-field">
  <label class="input-field__label" for="field">Лейбл</label>
  <div class="input-field__wrap">
    <input class="input-field__input" id="field" type="text" placeholder="...">
  </div>
</div>

<!-- С иконкой -->
<div class="input-field input-field--with-icon">
  <div class="input-field__wrap">
    <i class="input-field__icon">...svg...</i>
    <input class="input-field__input" type="text">
  </div>
</div>

<!-- Ошибка -->
<div class="input-field input-field--error">
  ...
  <span class="input-field__error-msg">Сообщение об ошибке</span>
</div>

Выпадающий список — Dropdown БЭМ: .dropdown

Нажмите на кнопку — откроется меню с анимацией. Клик вне — закрывается.

  • Редактировать
  • Дублировать
  • Удалить
  • Экспорт CSV
  • Экспорт PDF
  • Поделиться
<div class="dropdown">
  <button class="btn btn--secondary btn--md dropdown__trigger">
    Выбрать <span class="dropdown__arrow">▾</span>
  </button>
  <ul class="dropdown__menu">
    <li class="dropdown__item"><a href="#">Опция 1</a></li>
    <li class="dropdown__item"><a href="#">Опция 2</a></li>
  </ul>
</div>

<!-- JS: element.classList.toggle('dropdown--open') -->

Чекбоксы и радиокнопки БЭМ: .checkbox / .radio

Кастомные элементы управления с поддержкой состояний checked, focus, disabled.

Чекбоксы

Радиокнопки

<!-- Checkbox -->
<label class="checkbox">
  <input class="checkbox__input" type="checkbox" checked>
  <span class="checkbox__label">Подписаться на рассылку</span>
</label>

<!-- Radio -->
<label class="radio">
  <input class="radio__input" type="radio" name="group">
  <span class="radio__label">Вариант A</span>
</label>

Карточки — Card БЭМ: .card

Карточка с заголовком, изображением и текстом. Три размера для standalone-использования. Внутри grid — заполняет ячейку целиком.

Standalone-размеры

Small

Маленькая карточка для компактного контента.

Medium

Стандартный размер для большинства случаев использования.

Без изображения

Карточка без изображения — для текстового контента, уведомлений или сводной информации.

<div class="card card--md">
  <img class="card__image" src="..." alt="...">
  <div class="card__body">
    <h3 class="card__title">Заголовок</h3>
    <p class="card__text">Описание</p>
  </div>
  <div class="card__footer">
    <button class="btn btn--primary btn--sm">Читать</button>
  </div>
</div>

Сетка — Grid & Flexbox БЭМ: .grid / .flex

Grid-сетка с адаптивными колонками. На планшете — 2 колонки, на мобильном — 1.

Галерея карточек — grid--3

Карточка 1

Описание первой карточки в grid-сетке из трёх колонок.

Карточка 2

На планшете эти карточки перестроятся в 2 колонки.

Карточка 3

На мобильном — в одну колонку.

<!-- 3-колоночная сетка -->
<div class="grid grid--3">
  <div class="card">...</div>
  <div class="card">...</div>
  <div class="card">...</div>
</div>

<!-- Автоматическая сетка -->
<div class="grid grid--auto">...</div>

<!-- Flex -->
<div class="flex flex--between flex--gap-4">...</div>

Навбар — Navbar БЭМ: .navbar

Горизонтальное меню на десктопе. На мобильных (<768px) — скрывается, появляется кнопка-гамбургер. Живой пример — верхний навбар этой страницы. 

<nav class="navbar">
  <a href="#" class="navbar__brand">Logo</a>

  <!-- Hamburger (видно только на мобильных) -->
  <button class="navbar__toggle" id="navToggle">
    <span class="navbar__toggle-bar"></span>
    <span class="navbar__toggle-bar"></span>
    <span class="navbar__toggle-bar"></span>
  </button>

  <ul class="navbar__menu" id="navMenu">
    <li class="navbar__item">
      <a href="#" class="navbar__link navbar__link--active">Главная</a>
    </li>
    <li class="navbar__item">
      <a href="#" class="navbar__link">Компоненты</a>
    </li>
  </ul>

  <div class="navbar__actions">
    <button class="btn btn--primary btn--sm">Войти</button>
  </div>
</nav>

<script>
  const toggle = document.getElementById('navToggle');
  const menu   = document.getElementById('navMenu');
  toggle.addEventListener('click', () => {
    menu.classList.toggle('navbar__menu--open');
    toggle.classList.toggle('navbar__toggle--active');
  });
</script>
