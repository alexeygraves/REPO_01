# ReactShop

Учебное SPA на React с маршрутизацией, загрузкой данных из API и управлением состоянием через Context API.

## Запуск

```bash
npm install
npm run dev
```

Открыть http://localhost:5173

## Маршруты

| Путь         | Страница                              |
|-------------|---------------------------------------|
| `/`          | Главная — приветствие + список избранного |
| `/list`      | Каталог товаров (загрузка из API)    |
| `/list/:id`  | Детальная страница товара            |
| `/about`     | О проекте                             |

## Структура проекта

```
src/
├── context/
│   └── AppContext.jsx   # глобальное состояние: кэш списка, избранное, тема
├── components/
│   ├── Navbar.jsx       # навигация с NavLink и переключателем темы
│   └── ProductCard.jsx  # карточка товара в списке
├── pages/
│   ├── Home.jsx         # главная страница
│   ├── List.jsx         # каталог с загрузкой из API
│   ├── Details.jsx      # детали товара с кнопкой «Избранное»
│   └── About.jsx        # о проекте
├── App.jsx              # роутер
├── main.jsx             # точка входа
└── index.css            # стили (CSS-переменные, light/dark тема)
```

## Функционал

- **React Router v6**: `<Routes>`, `<Route>`, `<NavLink>` с активным классом, `useParams`, `useNavigate`
- **API**: данные загружаются из [DummyJSON](https://dummyjson.com/products) — бесплатный публичный REST API
- **Context API**: список товаров кэшируется (возврат на `/list` не делает повторный запрос); избранное хранится глобально
- **Обработка ошибок**: индикатор загрузки, сообщение об ошибке с кнопкой повтора
- **Тема**: светлая/тёмная, переключается кнопкой в навбаре
