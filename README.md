## Объект тестирования

Веб-приложение для банковских переводов (локальный сервис).  
Запуск: `http://localhost:9999`

## Структура репозитория

```
docs/
  test-cases.md       — 5 ручных тест-кейсов
  bug-report-001.md   — баг: принимается отрицательная сумма перевода
  bug-report-002.md   — баг: перевод принимается при сумме выше баланса
tests/
  test_transfer.py    — Selenium-тесты
  requirements.txt    — зависимости
.github/workflows/
  ci.yml              — GitHub Actions CI
```

## Запуск тестов локально

```bash
pip install -r tests/requirements.txt
pytest tests/ -v
```

Для запуска тестов необходимо, чтобы сервис был запущен на `http://localhost:9999`.

## CI

Сборка намеренно падает: тесты воспроизводят найденные дефекты.
