# BUG-002: Перевод выполняется при сумме выше доступного баланса

**ID:** BUG-002  
**Дата обнаружения:** 2026-05-23  
**Обнаружил:** Алексей Грейвс  
**Приложение:** Money Transfer, localhost:9999  
**Тест-кейс:** TC-004  

---

## Серьёзность (Severity)

**Critical** — приложение допускает уход баланса в минус без каких-либо ограничений или предупреждений.

## Приоритет (Priority)

**High**

---

## Описание

Серверная часть приложения не проверяет, достаточно ли средств на счёте перед выполнением перевода. Можно перевести сумму, превышающую доступный остаток, — операция принимается без ошибки, баланс становится отрицательным.

---

## Шаги воспроизведения

1. Открыть приложение `http://localhost:9999`
2. Авторизоваться (текущий баланс: 1000 руб.)
3. Перейти в форму перевода
4. В поле «Сумма» ввести: `5000`
5. Указать любой валидный счёт получателя
6. Нажать кнопку «Перевести»

## Ожидаемый результат

Отображается сообщение об ошибке: «Недостаточно средств на счёте». Перевод не выполняется, баланс не изменяется.

## Фактический результат

Перевод принят. Баланс счёта отправителя стал `-4000` руб.

---

## Окружение

- OS: Ubuntu 22.04
- Browser: Chrome 124
- URL: http://localhost:9999

## Вложения

Скриншот с отрицательным балансом после перевода — (при наличии прикрепить).
